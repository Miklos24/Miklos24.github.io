Latest Projects

Hated in the Nation

An analysis of the most loved (and hated) states in the US

New

For this project, I wanted to practice putting together a tech stack for a web app that is backended by some interesting data analysis. To that end, I decided to run a sentiment analysis on a sample of Tweets about the various states in the United States and produce a heatmap showing which states seemed to produce the most negative and positive sentiments in the Twitterverse. The project is backended by Flask, which uses Twitter's APIv2, pandas, and Natural Language Toolkit to do the analysis and stores the data in a PostgreSQL DB. This backend uses a REST API to communicate with the React front, which renders the data onto a heatmap of the United States and allows the user to query a specific date or sentiment.
